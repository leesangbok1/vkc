# Product Requirements Document (PRD)
# Viet K-Connect: 신뢰 기반 베트남인 Q&A 커뮤니티 플랫폼

---

## 1. Executive Summary

### 1.1 제품 비전
Viet K-Connect는 한국에 거주하는 베트남인들이 **24시간 내에 검증된 선배들로부터 신뢰할 수 있는 답변**을 받을 수 있는 Q&A 커뮤니티 플랫폼입니다.

### 1.2 핵심 가치 제안
> "한국생활 관련 정보 질문, 믿을 수 있는 선경험자가 24시간 내에 답변합니다"

---

## 2. 문제 정의

### 2.1 현재 상황
- 한국 거주 베트남인 **30만명** (근로자 14만 + 유학생 8만 + 기타)
- 주요 정보 채널: Facebook 그룹 (91.6% 사용)
- 정보의 신뢰성 문제로 인한 실제 피해 발생

### 2.2 핵심 문제점

| 문제 | 현재 상황 | 결과 |
|------|----------|------|
| **신뢰성 부족** | 익명 댓글, 검증 불가 | 잘못된 정보로 비자 문제, 금전 피해 |
| **정보 파편화** | 시간순 피드, 검색 어려움 | 같은 질문 반복, 정보 찾기 포기 |
| **응답 불확실** | 언제 답변 받을지 모름 | 긴급한 문제 해결 불가 |
| **언어 장벽** | 한국어 정보 이해 어려움 | 정보 접근성 제한 |

### 2.3 사용자 인사이트
> "페이스북에서 물어봤는데 다들 다른 얘기를 해요. 누가 맞는지 모르겠어요."
> - D-2 비자 유학생, 23세

> "E-7 비자 연장하려는데 서류가 뭐가 필요한지, 경험자 조언이 절실해요."
> - E-7 근로자, 28세

## 4. 솔루션 설계

### 4.1 핵심 기능

#### 🎯 Smart Matching System (스마트 매칭)
**문제 해결**: 무작위 답변 → 검증된 경험자 매칭

```
[질문 등록] → [AI 분석] → [5명 추천] → [선택/자동배정] → [24시간 내 답변]
```

- **매칭 기준**: 카테고리 + 비자 타입 + 경험 + 활동성
- **사용자 가치**: "내 상황과 비슷한 선배가 답변해줍니다"

#### ⏰ 24-Hour Guarantee (24시간 답변 보장)
**문제 해결**: 응답 불확실 → 시간 약속

- **시각화**: 타이머 색상 변화 (🟢→🟡→🟠→🔴)
- **자동 에스컬레이션**: 12시간 무응답시 추가 3명 할당
- **사용자 가치**: "내일 이 시간까지는 꼭 답변 받습니다"

#### 🏆 Trust Level System (신뢰도 시스템)
**문제 해결**: 익명성 → 투명한 신원 확인

```
Lv.23 🎖️ Nguyen Van A (한국 3년차 | E-7 삼성전자)
```

- **레벨**: 활동 기반 (답변, 채택, 좋아요)
- **인증**: 비자, 거주년차, 직장/학교
- **뱃지**: 활동, 신뢰, 전문 카테고리
- **사용자 가치**: "검증된 선배임을 한눈에 알 수 있습니다"

### 4.2 사용자 경험 흐름

#### 질문자 Journey
```
1. 홈 방문 (로그인 없이 탐색)
   ↓
2. 질문 작성 시도 → 소셜 로그인 유도
   ↓
3. 질문 등록 → 카테고리 자동 분류
   ↓
4. 5명 추천 받음 → 스와이프로 선택
   ↓
5. 24시간 내 답변 도착 → 알림
   ↓
6. 답변 평가 → 채택 → 공유
```

#### 답변자 Journey
```
1. 매칭 알림 수신 "당신의 경험이 필요합니다"
   ↓
2. 질문 확인 → 답변 작성
   ↓
3. 채택됨 → +50 XP, 레벨업
   ↓
4. 프로필에 시니어, 전문가 뱃지 획득
```

## 5. 디자인 원칙

### 5.1 Mobile-First
- **기준**: 360px 모바일 화면 우선
- **Thumb Zone**: 하단 80% 영역에 주요 기능
- **제스처**: 스와이프, 바텀시트, 플로팅 버튼

### 5.2 Trust by Design
- **실명 표시**: 모든 사용자 실명 공개
- **인증 뱃지**: 검증 정보 시각화
- **투명성**: 답변자 프로필 상세 공개

### 5.3 Speed & Simplicity
- **3-Tap Rule**: 모든 주요 기능 3번 탭 이내
- **로딩**: 1.5초 이내 첫 화면
- **온보딩**: 30초 이내 이해

## 8. 비즈니스 가치

## 10. 우선순위 및 로드맵

### 10.1 MVP (3주)
**목표**: 작동하는 프로토타입 with 핵심 기능

#### Must Have
- ✅ 소셜 로그인 (Facebook/Google)
- ✅ Q&A CRUD
- ✅ 5명 매칭 시스템
- ✅ 24시간 타이머
- ✅ 레벨/뱃지 시스템
- ✅ 모바일 최적화
- ✅ 카카오/이메일 알림 **질문할 때 이메일 카톡 주소 물어보는 박스 칸이 나오고 입력하면 알림해준다고 알려주는 창 생성 
#### Nice to Have
- ⏸️ AI 고도화 - 챗봇, 그외 부가 기능 에이전트
- ⏸️ 자동 번역 - 새로 생성되는 게시글 및 답변까지 모두 자동 번역
