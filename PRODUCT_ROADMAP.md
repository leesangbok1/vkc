# Product Roadmap 2025-2026
# Viet K-Connect - 제품 로드맵

## 🚀 Vision Statement
**"2026년까지 동남아 최대 한국 생활 정보 플랫폼으로 성장"**

한국에 거주하는 베트남인 30만명 중 50%가 활발히 사용하는 필수 플랫폼이 되어,
이후 태국, 필리핀 등 다른 동남아 국가로 확장합니다.

- **콘텐츠**: 30개 시드 Q&A

### 주요 마일스톤

```yaml

  - 데이터베이스 스키마 구현

  - 기본 검색 기능

```

#### M2: AI Features (2월)
```yaml

  - OpenAI GPT-3.5 연동

  - 태그 자동 생성

Week 7-8:
  - 전문가 매칭 알고리즘
  - 스마트 추천 시스템
```

#### M3: Beta Launch (3월)
```yaml

  - 실시간 알림 시스템
```
